This directory records the RLRMP-owned migration of selected legacy Feedbax
checkpoints into Feedbax schema-versioned model artifacts. Tracked files under
`migrated/` contain GraphSpec companions, Feedbax `ModelArtifactManifest`
records, and migration summaries; bulk role-addressed array stores live in the
mirrored `_artifacts/b41c940/migrated/` tree and are intentionally not tracked.
