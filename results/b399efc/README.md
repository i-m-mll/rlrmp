This experiment tests whether faster reaching is better controlled by a
movement-locked, fixed-duration position-error ramp than by the prior
full-trial power-law schedule, while keeping the saturated pre-go output penalty
that suppressed anticipation in the preceding matrix. The run sequence is a
1k-iteration stability smoke test of the previous best full-trial power-law
cell with `nn_output_pre_go=100`, followed by five 12k warmup-only ramp cells
with zero target-on/pre-movement position and velocity cost.
